# Sistema de notas

Este es un programa en Python para registrar y gestionar las notas de los estudiantes de manera sencilla. Ideal si quieres llevar un control rápido de las materias y sus calificaciones.

## Características

- **Registro y Autenticación:** Sistema de usuarios "profesores" con almacenamiento en diccionarios.
- **Dashboard de Usuario:** Panel para ingresar las notas 
- **Persistencia Temporal:** Gestión de datos estructurada mediante listas y diccionarios.

## Tecnologías Utilizadas

- **Lenguaje:** Python
- **Versionamiento:** Git & GitHub (Metodología Git Flow)
- **Entorno:** Virtualenv

##  Requisitos Previos

Asegúrate de tener instalado Python 3.10 o superior.
